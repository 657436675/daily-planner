# daily-planner 🎯

> 目标驱动的每日自律系统 — 把任何目标拆成每天能执行的动作，用游戏化追踪 + AI日常督促保证落地

## 这是什么？

daily-planner 是一个 [WorkBuddy](https://www.codebuddy.cn) Skill 插件，帮用户把大目标拆解成每天能执行的小动作。不只是生成一个任务清单——它帮你诊断现状、确定方向、拆解阶段、定制每日时间表，然后用游戏化追踪器 + 可选的AI日常督促来保证落地。

## 核心功能

- **🧠 初始化访谈** — 加载skill后AI主动追问，帮你理清现状和目标
- **📋 阶段拆解** — 根据你的紧迫程度拆成2-3个可执行阶段
- **⏰ 每日时间表** — 根据你的作息习惯定制每日任务安排
- **🎮 游戏化追踪器** — 完成任务赚XP、升级、解锁成就
- **🤖 可选AI督促** — 早晚报 + 周复盘，AI主动问你进展（完整版）

## 安装

### 前提条件

- 已安装 [WorkBuddy](https://www.codebuddy.cn)（AI桌面助手）

### 安装步骤

1. 下载本仓库所有文件
2. 将文件放到 `~/.workbuddy/skills/daily-planner/` 目录下，保持目录结构：

```
~/.workbuddy/skills/daily-planner/
├── SKILL.md                  # skill主文件（必需）
├── tracker-template.html     # 追踪器HTML模板（必需）
├── templates/                # 模板配置目录（可选，缺了自定义模板就用默认）
│   ├── 1-创业转型.json
│   ├── 2-技能深耕.json
│   ├── 3-考试冲刺.json
│   ├── 4-身材管理.json
│   ├── 5-副业启动.json
│   └── 6-自媒体IP.json
```

3. 在 WorkBuddy 对话中输入 `daily-planner` 或 `每日规划` 即可启动

## 追踪器

生成的追踪器是一个纯HTML文件，浏览器直接打开，不需要安装任何东西：

- **XP经验值** — 完成任务赚XP，不同任务XP不同
- **等级系统** — 累计XP达到阈值自动升级，每套模板有专属等级名
- **成就徽章** — 达成特定条件自动解锁（连续打卡、早起鸟、阶段完成等）
- **每日重置** — 每天0点任务勾选状态重置，XP和等级永久保留
- **数据持久化** — 所有数据保存在浏览器 localStorage，同一浏览器不丢失
- **自定义增删** — 随时增删改任务，追踪器完全可控

## 自定义模板构建器

如果你的目标不在6套预置模板里，选"自定义"。AI会追问你5个方面：

1. **目标定义** — 你想达成什么？
2. **现状评估** — 离目标有多远？优势/短板？
3. **时间投入** — 每天能拿出多少时间？
4. **紧迫程度** — 压力有多大？
5. **动机来源** — 为什么想做？

然后自动帮你生成专属的等级线、成就徽章、每日任务模板和阶段规划。

> 所有内容都可以后续调整，模板不是锁死的。

## FAQ

**需要付费吗？** Skill本身免费。自动化功能需要 WorkBuddy 付费版。

**数据安全吗？** 追踪器数据全保存在浏览器本地，不上传任何服务器。

**换了电脑怎么办？** 当前数据绑定浏览器。后续计划做成多用户网站解决此问题。

**可以同时用多套模板吗？** 一次生成一套追踪器，但可以随时找AI调整任务内容。
